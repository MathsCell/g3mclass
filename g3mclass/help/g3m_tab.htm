Tabs' content

The content of G3MClass is organized in tabs and sub-tabs. The meaning of main tabs is following:

  • 'Welcome' is active at G3MClass start and briefly explains how to start a work with G3MClass.
  • 'Data' contains input data in a spreadsheet form
  • 'Parameters' contains controls for modeling and graphics parameters
  • 'Model plots' shows Gaussian plots learned from test samples
  • 'Test class' is a spreadsheet with classification results of test samples. Results of each bio-marker is given in its own sub-tab
  • 'Ref class' idem for reference samples
  • 'Query class' idem for query samples
  • 'Heatmaps' shows classification results in heatmap form, one heatmap per ref-, test- and each query-samples. Heatmaps are only shown if 'id' columns are present in the input data file.
