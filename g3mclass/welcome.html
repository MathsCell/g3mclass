Welcome to g3mclass!

g3mclass is a software for Gaussian Mixture Model for Marker Classification.

Usage:

  1. Open a data file: File > Open > ...(choose some file, let call it data.tsv) data.tsv must contain duplicate or triplicate columns relative to some biomedical markers. If we have a marker 'geneA' then we should have following columns: geneA (ref), geneA (test) and optionally geneB (query). The first column geneA (ref) contains reference sample. Data can be visualized in the tab Data next to this one.
  2. Data from columns (test) are used to learn Gaussian mixtures where (ref) sample is imposed with its mean and sd values. Adjust model parameters in tab Parameters if needed and hit Lear model button. The learned model content can be viewed in tabs Model and Model plots.
  3. Data from all columns (ref), (test) and (query) (if present) are automatically classified. The classification results can be examined in tabs Test class, Ref class,Query class and Heatmaps (if id columns were provided in the input data file).
  4. If results are satisfactory, they can be saved in a ZIP archive (File > Save)
