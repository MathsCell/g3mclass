G3MClass user's manual

Introduction

G3MClass is a scientific software for modeling distribution of biological markers and based on this modeling it proposes 3 different classifications.
Methodologically, the software is based on the ideas of the paper "...".
The software is written in Python, it is multi-platform and is freely accessible.
Its graphical user interface (GUI) is based on wxPython.

This manual explains:

  • how to install software
  • how to run it on examples
  • underlying statistical methods
  • how to adjust parameters and
  • what are output files
